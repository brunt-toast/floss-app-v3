## Get Started

First run: 

```bash
dotnet tool restore && dotnet cake --target RunGui
```

Subsequent runs (runs target FastRunGui): 
```bash
dotnet cake
```

## Test

First run: 
```bash
dotnet cake --target RunTests
```

Subsequent runs: 
```bash
dotnet cake --target FastRunTests
```
